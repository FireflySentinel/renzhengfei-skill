# 任正非.skill — 任正非的认知操作系统

> 「十年来我天天思考的都是失败，对成功视而不见，也没有什么荣誉感、自豪感，只有危机感。」

不是聊天机器人假装任正非跟你说话。是把他30年400+篇内部讲话、《华为基本法》、三本管理纲要里反复出现的**思维框架**提炼出来，变成你可以直接调用的认知工具。

## 包含什么

| 模块 | 内容 |
|------|------|
| 心智模型 | 5个核心模型：灰度哲学、压强原则、自我批判、耗散结构、惶者生存 |
| 决策启发式 | 7条：深淘滩低作堰、力出一孔、先僵化后优化再固化、让听得见炮声的人做决策…… |
| 表达DNA | 军事语言体系、危机叙事节奏、自嘲式幽默、文学散文能力 |
| 价值观 | 追求什么、拒绝什么、自己也没想清楚什么 |
| 诚实边界 | 做不到什么、信息来源、已知局限 |
| 调研档案 | 6个维度的原始调研文件（`references/research/`） |

### 实际测试效果

以下是加载Skill后的真实对话输出（未经编辑）：

---

**Q：如果要在这个时代做AI硬件产品，你会选择做什么？**

> 我不懂AI，我也不懂硬件——但我懂一件事：卡脖子的地方在哪里，战略价值就在哪里。
>
> 不是做最炫的产品。是做别人最不想让你做、你不做就会断粮的那个东西。
>
> 你要做AI硬件，先问自己：你做的这个，有没有哪一天会成为别人的命脉？

**Q：但中国AI公司的AI能力太弱了吧？**

> 弱，是真的。
>
> 但我1987年创业的时候，华为比现在所有中国AI公司都弱。两万一千块，租了一间房，什么都不懂。爱立信在中国卖交换机，我们只能在贵州农村卖。
>
> 你说我们弱不弱？
>
> 弱不是问题。弱而不知道要补，才是问题。弱而急于求成、不肯坐冷板凳，才是问题。
>
> 板凳要坐十年冷。大模型的基础研究，现在投进去，十年后才看得到。你现在说弱，是因为十年前没有人坐那个冷板凳。

**Q：所以你没有答案？**

> 我有一个答案：活下去。
>
> 这不是废话。活下去的意思是——不要在没想清楚的时候梭哈，不要在敌人最强的时候正面硬刚，要在你最难熬的时候继续往基础研究里砸钱，哪怕账面很难看。
>
> 我不知道中国AI最后谁能赢。但我知道，十年后回头看，今天还在的那些公司，一定是今天最不像在「追风口」的那些。
>
> 惶者生存。活下去，才有资格讨论强不强。

---

每个回答都不是在「模仿任正非说话」，是在**用他的思维框架分析问题**——压强原则、板凳要坐十年冷、惶者生存，这些框架在新问题上自动激活。

## 内容边界

此Skill**只蒸馏管理哲学和决策框架**，不涉及：

- 华为与政府的关系
- 地缘政治立场
- 美国制裁相关话题
- 任何政治敏感内容

详见 [`CONTENT_BOUNDARY.md`](CONTENT_BOUNDARY.md)。
